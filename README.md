## Didáctica de la Enseñanza de la computación
<p>
De GitHub aprendí que es una herramienta utilizada por la mayoría de desarrolladores. Esta herramienta permite trabajar el código fuente de una aplicación desde la nube (un respositorio) permitiendo la colaboración entre varias personas. Esto permite que un equipo de trabajo pueda estar trabajando sobre un mismo código. 

Github cuenta con herramientas como GIT que nos permite llevar un control de versiones, es decir, controlar quienes hacen modificación, en qué espacio específico lo hacen y cuándo lo hacen.
</p>
